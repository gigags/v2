giga.gs

API

UPLOAD REQUEST

The base URL to upload your file is https://api.giga.gs/upload provided with the public or your private token.

The token can be use in HEADER / POST / GET parameter

BASIC EXAMPLE

$token as variable.

$ curl -F file=@"/path/of/your/file.mp4" api.giga.gs/upload?token=$token
$ curl -F file=@"/path/to/your/file.mp4" -H 'token: $token' api.giga.gs/upload

CLICK HERE TO VIEW

The response is as is. The response success is boolean and if true the url will be replaced with a direct link of the uploaded file.
